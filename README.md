# **itu-helper/chrome-extension**

## **Ne İşe Yarar?**

İTÜ'nün dağınık web sitelerini, ITU Helper hizmetlerini ve diğer öğrenci sitelerini bir araya getiren ve kolay erişim sağlayan bir Chrome eklentisidir.


## Kullanım

Eklenti kurulduktan sonra, Chrome tarayıcınızın sağ üst köşesindeki eklenti simgesine tıklayarak ITU Helper'ı açabilirsiniz. Açılan pencerede istediğiniz ITU web sitesine tıklayarak hızlıca erişebilirsiniz.
