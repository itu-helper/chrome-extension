# **itu-helper/chrome-extension**

## **Ne İşe Yarar?**

İTÜ'nün dağınık web sitelerini, ITU Helper hizmetlerini ve diğer öğrenci sitelerini bir araya getiren ve kolay erişim sağlayan bir Chrome eklentisidir. Eklentiye [bu](https://chromewebstore.google.com/detail/itu-helper/cinnpmhpiddpbjmmdmlncnlfhjanigkl?hl=en&authuser=0) linkten ulaşabilirsiniz.

## Kullanım

[Eklenti]((https://chromewebstore.google.com/detail/itu-helper/cinnpmhpiddpbjmmdmlncnlfhjanigkl?hl=en&authuser=0)) kurulduktan sonra, Chrome tarayıcınızın sağ üst köşesindeki eklenti simgesine tıklayarak ITU Helper'ı açabilirsiniz. Açılan pencerede istediğiniz ITU web sitesine tıklayarak hızlıca erişebilirsiniz.

## Dosya Yapısı

Eklentiye dair kodların tamamı `./src/` klasörünün içinde toplanmıştır. Projenin tepesindeki `./css/` ve `./webfonts/` klasörleri, _FontAwesome_ paketi uyumluluğu için kullanılıyor.
